# concurrentFileHandler
Из большого текстового файла (>10 МБ), который содержит в себе символы английского алфавита от “a” до “z” (строчные буквы) в рандомной последовательности, требуется вычитать в 5 потоков все содержимое, провести циклический сдвиг символов (например, символ “a” нужно заменить на “b”, “b” на “c” , ….. , “z” на “a”) и записать преобразованные данные в другой файл. Название входного и выходного файла должны подаваться на вход вашей программе.
